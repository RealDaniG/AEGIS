1. Clone the repository:
   ```bash

   cd MetatronV2-Open-A.G.I-
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the system:
   ```bash
   cd Metatron-ConscienceAI
   START_SYSTEM.bat  # Windows
   # or
   ./START_SYSTEM.sh  # Linux/macOS
   ```
